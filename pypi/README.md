# fastapi-init

Scaffold a production-ready FastAPI project in seconds.

## Install

```bash
pip install fastapi-init
```

## Usage

```bash
fastapi-init my-project
# or scaffold into the current directory
fastapi-init .
```

The CLI will guide you through selecting a database, ORM, auth provider, Docker support, and more.

## Source

Built with Go. Binaries are downloaded from [GitHub Releases](https://github.com/markryangarcia/fastapi-init/releases) on first run and cached at `~/.cache/fastapi-init`.
